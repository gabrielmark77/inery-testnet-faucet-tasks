### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to run

Change directory to ```gianluigi```

```shell
cd ~/inery-testnet-faucet-tasks/gianluigi
```

Copy .env-sample and rename it to .env

```shell
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
npm run solution
```
